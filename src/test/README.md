# Test Module

Testing utilities and fixtures.

## Structure

- `setup.ts` - Test environment setup
- `fixtures/` - Test data fixtures
- `mocks/` - Mock implementations
- `helpers/` - Test helper functions

## Running Tests

```bash
# All tests
npm test

# Unit tests
npm run test:unit

# Integration tests
npm run test:integration

# Coverage
npm run test:coverage
```
